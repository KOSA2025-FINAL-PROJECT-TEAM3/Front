## 🚀 피그마에 Import하기

### 1단계: 피그마 Desktop 앱 열기
⚠️ **중요**: 웹 버전이 아닌 Desktop 앱 필수!

### 2단계: 플러그인 실행
1. 피그마에서 `Plugins → Development → 실버케어 JSON Importer` 클릭
2. 플러그인 UI가 나타남

### 3단계: JSON 복사
```bash
# 전체 파일 복사 (추천!)
cat figma-exports/all-screens-with-flows.json
```
- **전체 내용 복사** (Ctrl+A → Ctrl+C)

### 4단계: Import 실행
1. 플러그인 UI에 **붙여넣기** (Ctrl+V)
2. **🚀 Import Multiple** 버튼 클릭
3. ⏳ 잠깐 기다리면...

## ✨ 결과 확인

### 📊 생성되는 화면 (총 8개)
1. **로그인** - 이메일/비밀번호 입력
2. **회원가입** - 사용자 정보 입력
3. **역할 선택** - 시니어 vs 보호자 선택
4. **시니어 대시보드** - 약 복용 체크리스트
5. **보호자 대시보드** - 가족 모니터링
6. **약 관리** - 약 목록 및 관리
7. **약 등록** - 새 약 추가 폼
8. **가족 관리** - 가족 구성원 관리

### 🔗 프로토타입 플로우 (총 15개 화살표)

#### 인증 플로우
```
로그인 → 역할 선택
회원가입 → 역할 선택
로그인 ↔ 회원가입
```

#### 역할 선택 플로우
```
역할 선택 → 시니어 대시보드 (시니어 선택 시)
역할 선택 → 보호자 대시보드 (보호자 선택 시)
```

#### 메인 네비게이션
```
시니어 대시보드 → 약 관리
시니어 대시보드 → 가족 관리
보호자 대시보드 → 약 관리
보호자 대시보드 → 가족 관리
```

#### 약 관리 플로우
```
약 관리 → 약 등록 (+ 약 추가 버튼)
약 등록 → 약 관리 (저장 버튼)
약 등록 → 약 관리 (뒤로가기)
```

#### 홈 복귀
```
약 관리 → 시니어 대시보드 (홈 버튼)
가족 관리 → 시니어 대시보드 (홈 버튼)
```

## 🎨 화살표 확인 방법

### 방법 1: 프로토타입 모드
1. 피그마 오른쪽 상단 **프로토타입 탭** 클릭
2. 화면 선택하면 → **파란색 화살표** 보임!
3. 화살표 클릭하면 연결된 화면 확인 가능

### 방법 2: 프레젠테이션 모드
1. 화면 선택 후 **▶️ 재생 버튼** 클릭
2. 실제로 클릭하면서 화면 전환 테스트
3. ESC로 나가기

## 🔧 문제 해결

### "Font not available" 경고
→ [Inter 폰트](https://fonts.google.com/specimen/Inter) 설치 필요
→ 또는 피그마에서 다른 폰트로 변경

### 플러그인이 안 보임
→ **피그마 Desktop 앱** 사용 중인지 확인 (웹 버전 안됨)

### "Invalid JSON structure"
→ JSON 파일 **전체** 복사했는지 확인
→ 파일 시작: `{` / 파일 끝: `}`

### 화살표가 안 보임
→ **프로토타입 탭**에서 확인 (디자인 탭에서는 안 보임)

## 💡 Tip

### 피그마에서 수정하기
- 모든 화면은 **Auto Layout** 적용됨
- 색상, 간격, 텍스트 자유롭게 수정 가능
- 컴포넌트로 변환하여 재사용 가능

### 화살표 수정하기
- 프로토타입 모드에서 화살표 클릭
- 트랜지션 효과 변경 가능 (현재: DISSOLVE 0.3초)
- 새로운 연결 추가 가능

### Export하기
- 개별 화면 PNG/SVG Export 가능
- 전체 프로토타입 공유 링크 생성 가능
